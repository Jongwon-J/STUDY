            - 변수분리형 미분방정식 : 동일 변수끼리 분리하여 두는 것
            
            $$
            g(y)dy = f(x)dx
            $$
            
            **< Tip >**
            
            - dy, dx가 분자에 있어야 적분 가능하다.
            - 로그일 경우에는 적분 상수를 lnC로 두고, 지수 형태 (e) 일 경우에는 앞 계수에 C를 붙여 적분 상수를 표현한다.
            
            **[ section 1.3 : 선형 미분방정식 ]**
            
            - 1계 선형 미분방정식의 형태
            
            $$
            dy/dx + a(x)y = f(x)
            $$
            
            - 동차 vs 비동차 선형 미분방정식의 차이
                - f(x) : 0 → 동차 선형 미분방정식
                - f(x) : 살아있음 → 비동차 선형 미분방정식
            - 비동차 선형 미분방정식의 풀이 공식에서의 적분 인자를 도입하는 이유
                
                → 정해진 적분인자를 두어야 곱의 미분 형태로 적분이 가능하기 때문이다.
                
            
            **[ section 1.5 : 완전미분방정식 ]**
            
            - 완전미분방정식의 형태
            
            $$
            M(x,y)dx + N(x,y)dy = 0
            $$
            
            1. 완전 미분 가능할 경우 ( 판별 조건 : M(x,y)에 대해 y 편미분 = N(x,y)에 대해 x 편미분 )
            
            → f(x,y) = M(x,y)에 대해 x 편적분 + N(x,y)에 대해 y 편적분 = c
            
            1. 완전 미분 불가능할 경우
            
            → 적분 상수를 정해진 식을 통해 구한 후 M(x,y)와 N(x,y)에 곱해 완전 미분 가능 형태를 만들고 (1)의 방법을 시행한다.
